# Excel Data Analytics & Transformation Portfolio

Welcome to my data portfolio. This repository showcases my practical skills in taking raw, unorganized, or multi-source datasets and using Excel and Power Query to clean, consolidate, and model them for operational business use. These projects were completed as part of a self-guided upskilling initiative, where I followed step-by-step technical tutorials to learn and apply foundational data transformation workflows.
---

## 📊 1. Costco Branch Data Consolidation
* **Business Context:** Regional sales data scattered across multiple individual branch files.
* **The Solution:** Developed a data pipeline using Power Query to ingest separate branch datasets, clean inconsistent formatting, eliminate duplicates, and compress them into a single unified master datasheet.
* **Core Skills:** Multi-file ingestion, ETL (Extract, Transform, Load), data normalization.

## 🍔 2. McDonald's Performance Dashboard
* **Business Context:** Raw operational or nutritional data requiring visual synthesis.
* **The Solution:** Transformed flat data rows into a dynamic dashboard layout, utilizing structured organization to highlight key business performance metrics at a glance.
* **Core Skills:** Dashboard design, data aggregation, visual hierarchy.

## 🚗 3. Automotive Inventory Management
* **Business Context:** Managing high-volume asset tracking with fluctuating stock levels.
* **The Solution:** Structured a raw car inventory dataset to track availability, model distributions, and stock statuses cleanly.
* **Core Skills:** Inventory modeling, data filtering, data integrity tracking.

## 💰 4. Corporate Payroll Processing
* **Business Context:** Handling sensitive operational employee hours, rates, and compensation structures.
* **The Solution:** Built a structured payroll calculation ledger ensuring clean logic flow across employee lists, standardizing data formats for accurate reporting.
* **Core Skills:** Logical functions, structural data auditing, payroll formatting.

## 📝 5. Academic Grade Book Tracker
* **Business Context:** Tracking multi-weighted performance records across cohorts.
* **The Solution:** Engineered a clean, formula-ready grade book template that aggregates raw scores and structures student records for seamless evaluation.
* **Core Skills:** Conditional organization, data formatting, cell dependency management.

## 🧼 6. Core Data Cleaning Foundations
* **Business Context:** Raw, messy public datasets containing structural human errors.
* **The Solution:** A dedicated sandbox proving core data hygiene skills: fixing broken text strings, handling missing values, splitting merged columns, and standardizing data types.
* **Core Skills:** Text manipulation, data type auditing, error handling.
