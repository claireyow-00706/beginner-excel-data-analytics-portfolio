# Costco Branch Data Consolidation Project

## Project Overview
This project demonstrates how to automate the consolidation of scattered business data. I took separate, raw sales and operational datasets from multiple Costco branches and used Excel Power Query to clean, transform, and compress them into a single, unified master datasheet.

## Core Technical Skills Applied
* **Data Ingestion:** Combining multiple source files from different branches into a single data pipeline.
* **Power Query ETL:** Extracting raw data, transforming inconsistent formatting, removing duplicates, and loading it into a clean layout.
* **Data Normalization:** Preparing regional datasets for seamless, centralized business reporting.

## The Workflow
1. **Extract:** Gathered individual branch files representing different Costco locations.
2. **Transform:** Used Power Query to align column headers, clean up data mismatches, and filter out noise.
3. **Load:** Consolidated the rows into a streamlined, high-performance master sheet ready for analysis.
